# AE 帮助信息

获取 AE 插件中所有可调用的技能、命令和代理的完整列表。

## 执行方式

**必须调用 `ae-help` 工具**，将结果原样返回给用户。

- 无参数：调用 `ae-help` 工具，不传入 `query`
- 有参数：调用 `ae-help` 工具，将用户输入传入 `query`

**禁止手动列举或扫描文件**，列表内容只能来自 `ae-help` 工具的返回值。

## 输出格式

直接将 `ae-help` 工具返回的 Markdown 文本输出给用户，不做任何修改、精简或二次加工。
