# AE Plan Review

目标：
- 对计划文档进行多角色审查
- 复用 `ae:document-review` 的审查角色族，对计划文档使用计划专属分类
- 使用 `ae-review-contract` 计算审查团队
- 确认实现顺序、范围、测试场景和恢复路径是否足够清晰
